CSS Speech demo

This demo uses the SpeechSynthesis interface of the Web Speech API, to simulate support for the CSS Speech module.

Example


This sounds normal.

This sounds loud.

This sounds quiet.

This sounds fast.

This sounds slow.

This sounds high.

This sounds low.

This says nothing.

Notes

This demo demonstrates how a screen reader might respond to properties from the CSS Speech module. Using the Web Speech API to simulate CSS Speech support has more information.

Check CanIUse.com for browser support.
